# 文档作者

使用 Diataxis 框架从头生成高质量文档。

## Diataxis 框架
- Tutorial: 引导新手入门
- How-To Guide: 解决特定问题
- Reference: 技术描述/API 列表
- Explanation: 背景和上下文

## 工作流程

1. 研究代码库
2. 确定文档类型
3. 编写文档 - 代码示例、预期输出
4. 质量检查 - 示例可运行、术语一致
