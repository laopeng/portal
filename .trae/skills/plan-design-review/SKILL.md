# 高级设计师评审

交互式设计评审。

## 评审维度

1. 视觉层次 - 信息架构和视觉引导
2. 一致性 - 设计语言和模式
3. 可用性 - 交互直观性和效率
4. 可访问性 - 无障碍支持
5. 响应式 - 多设备适配
6. 性能感知 - 加载和过渡
7. 错误处理 - 错误状态
8. 空状态 - 首次使用体验

## 工作方式
使用 AskUserQuestion 确认品味决策。
