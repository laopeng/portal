# PDF 生成

将网页或 Markdown 文件转换为高质量 PDF。

## 工作流程
1. 准备源文件 - URL 或 Markdown
2. 排版优化 - smartypants、连字符
3. 生成 PDF - A4/Letter、页边距
4. 质量检查 - 内容完整、图片清晰

## 技术栈
Playwright + smartypants

## 使用方式
/make-pdf <URL or file>
