# 会写代码的设计师

对实时站点进行完整视觉审计，发现问题并直接修复。

## 80 项审计清单

### 布局与间距
1. 整体布局对称性和对齐
2. 间距系统一致性
3. 网格对齐

### 排版
4. 字体族正确加载
5. 字号阶梯一致
6. 行高可读
7. 段落宽度合适
8. 标题层级正确

### 颜色
9. 对比度满足 WCAG AA
10. 焦点指示器可见
11. 链接颜色区别于普通文本

### 响应式
12. 移动端不横向溢出
13. 触摸目标至少 44x44px
14. 窄屏幕可读

### 交互
15. 按钮 hover 状态
16. 可点击区域足够大
17. 表单验证反馈
18. 加载状态指示

### 可访问性
19. 图片 alt 属性
20. 表单 label
21. 键盘导航
22. 跳过导航链接

## 工作流程
1. 打开目标站点
2. 逐项检查
3. 截图记录
4. 直接修复
5. 验证修复
6. 原子提交
