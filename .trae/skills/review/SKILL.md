# 高级工程师代码评审

找出通过 CI 但在生产中会爆炸的 bug。

## 检查清单

1. 并发安全 - 竞态条件、死锁、共享状态
2. 安全漏洞 - SQL 注入、XSS、认证绕过
3. 错误处理 - 所有操作都有错误处理
4. 边界情况 - null/undefined、空集合、极限值
5. 测试覆盖 - 关键路径有测试
6. 性能 - N+1 查询、内存泄漏、阻塞操作

## 修复策略
- 能确定的自动修复
- 需要背景的标记说明
- 完整性缺口列出缺失项
