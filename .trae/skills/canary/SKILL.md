# SRE 金丝雀监控

部署后的持续监控，确保新版本在生产环境中稳定运行。

## 监控维度

1. 控制台错误 - JS 错误、资源加载失败
2. 性能回归 - 页面加载、Web Vitals、API 响应
3. 页面故障 - 白屏、功能不可用、样式错乱

## 监控循环
- 每 30 秒检查一次
- 持续 5-10 分钟
- 发现问题立即报告

## 告警阈值
- 错误率增加 > 1%
- 加载时间增加 > 20%
- API 响应增加 > 50%
- 任何 5xx 错误

## 输出
部署后健康报告、异常记录、回滚建议。
